# ミュージシャンポートフォリオサイト

参考サイト（Studio Narrative）を基にしたミュージシャンのポートフォリオサイトです。

## 機能

- **ランディングページ**: プロフィール、写真、ワークスへのリンク
- **ワークス一覧ページ**: 作品の一覧表示
- **ワークス詳細ページ**: 参考サイト風のデザインで各作品の詳細を表示
- **Aboutページ**: アーティストについて
- **Contactページ**: お問い合わせ情報

## 技術スタック

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- React

## セットアップ

1. 依存関係のインストール:
```bash
npm install
```

2. 開発サーバーの起動:
```bash
npm run dev
```

3. ブラウザで [http://localhost:3000](http://localhost:3000) を開く

## カスタマイズ

### ワークスデータの編集

`app/works/[id]/page.tsx` の `worksData` オブジェクトを編集して、作品情報を追加・変更できます。

### スタイリング

Tailwind CSSを使用しているため、各コンポーネントのクラス名を編集することでデザインをカスタマイズできます。

### 画像の追加

`public/works/` ディレクトリに画像を追加し、ワークスデータの `image` プロパティで参照してください。
